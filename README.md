Todo App
The Todo app uses the following technologies :
* MongoDB
* Express.js
* Node.js
* express-handlebars
* method-override
* connect-flash
* express-session
* mongoose
* bcryptjs
* passport

To install nodemodules run command - npm install
To start app run command - npm start
